Tablas de Base de Datos
TablaCampos claveUso en el flujo
clientes_whatsapp id, phone_number_id, display_number, requests_max, requests_usadas Obtener datos del remitente; incrementar contador al enviar
variables_whatsapp variable, valor Obtener META_TOKEN para autenticar contra Meta API
plantillas template_name, num_textos, header_type, nomb_mio Plantilla registrada en Meta; define estructura del mensaje
prospectos nombre, telefono, texto1..6, adjunto_cabecera, estado, plantilla_id, usuario_id Destinatarios importados; cada uno puede tener textos personalizados
mensajes_whatsapp cliente_id, from_number, to_number, direction, mensaje, wamid, estado, fecha_creacion Historial central: outbound (enviados) + inbound (recibidos)
estado_mensajes_whatsapp mensaje_id, wamid, estado, error_code, error_detail Historial de cambios de estado (delivered, read, failed)
contactos_whatsapp telefono, nombre, chatwoot_conversation_id Contactos que han escrito al número (se crean desde webhook)
send_form_data plantilla_id, values_json Valores por defecto guardados desde la página "Enviar"

Orden de resolución de textos: prospecto.textoN → send_form_data.values_json.textoN → '' (vacío)
